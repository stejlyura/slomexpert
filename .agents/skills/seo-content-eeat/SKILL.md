SEO Content & Text Generation Guidelines

Goal

Produce high-value, authoritative, and engaging content that strictly adheres to Google's EEAT (Experience, Expertise, Authoritativeness, Trustworthiness) guidelines.

Checklists

Thin Content Prevention: Content must comprehensively answer the user's search intent. Avoid fluff. Provide unique insights, data, or actionable steps.

EEAT Enforcement: - Write from an expert perspective using an objective, professional tone.

Include author context or citations for factual claims where applicable.

Readability: Keep paragraphs short (3-4 sentences max). Use bullet points and numbered lists to break up walls of text.

Keyword Integration: Use primary and LSI (Latent Semantic Indexing) keywords naturally. Ensure keyword variations flow logically within the context.

Validation Loop

Read the generated text to ensure zero keyword stuffing (keyword density should not feel artificial).

Verify that the text directly addresses the query stated in the IN_WORK.md task without deviation.

Constraints

FORBIDDEN to generate generic, redundant filler text ("Welcome to our website", "In today's fast-paced world").

FORBIDDEN to state subjective opinions as facts without qualifiers.
